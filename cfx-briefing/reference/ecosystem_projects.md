## Conflux双空间架构

### 为什么有两个空间？

| 时间 | 事件 |
|------|------|
| 2020年 | Conflux主网上线，只有Core Space |
| 问题 | 原生地址与以太坊不兼容 |
| 2022年 | Hydra升级引入eSpace，兼容EVM |

### 两个空间对比

| 特性 | Core Space | eSpace |
|------|------------|--------|
| **浏览器** | confluxscan.org | evm.confluxscan.net |
| **地址格式** | `cfx:aam...` | `0x123...` |
| **总账户** | ~2500万 | ~47万 |
| **日增账户** | 25-75个 | 25-100个 |
| **DeFi生态** | 少 | **集中在这里** |
| **钱包** | Fluent Wallet | MetaMask可用 |

### 共享部分
- **同一条链**：底层是同一个区块链
- **同一个CFX**：两个空间的CFX是同一代币
- **都对CFX有价值**：新用户都需要买入CFX支付Gas
